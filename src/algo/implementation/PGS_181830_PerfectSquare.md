## 정사각형으로 만들기

### 문제 요약

- 정수 2차원 배열 arr가 주어짐
- arr의 행 수와 열 수 중 더 큰값을 size라고 할때 size x size 정사각형 배열을 만들어 반환
- 원래 배열의 값은 좌상단부터 그대로 복사하고, 부족한 칸은 0으로 대채한다.

### 접근 방식

- Math.max()로 rows와 cols의 값을 비교해서 정사각형인지 아닌지 체크 하면서 접근 

### 왜 이런 방식으로 풀었나?

- 자바 배열은 크기 변경이 불가능하므로, 새 정사각형 배열을 만들고 복사하는 방식이 가장 안전하고 직관적이기 때문에
- 0으로 자동 초기화되므로 패딩 처리가 간단
