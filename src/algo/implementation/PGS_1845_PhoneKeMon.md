## 폰켓몬

### 문제 요약

- nums 배열에 폰켓몬 종류 번호가 들어가있다.
- 이중 n/2마리만 선택 할 수 있다.
  - num.length는 항상 짝수
- 선택한 n/2마리에 최대한 많은 종류의 폰켓몬을 포함해야한다.

### 접근 방식

- 배열 속 같은 수를 최소화 하기위해 set을 사용해 중복을 제거해준다.
- n/2마리내에서 중복이 아닌 최대한 많은 수를 선택해야하기 때문에
- set.size()와 n/2마리의 수를 Math.min()을 적용해서 더 적은 수의 값을 추출하였다.

### 왜 이런 방식으로 풀었나?

- 많은 수의 배열이 담겨있다 한들 최소한으로 뽑아야할 수는 정해져 있다.
- 그래서 set의 수와 n/2로 계산된 두개의 수만 비교하여 값을 뽑아내는게 가장 간단한 방법이라 생각했다.
