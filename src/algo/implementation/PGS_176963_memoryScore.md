## 추억 점수

### 문제 요약

- 그리워하는 사람들의 이름이 담긴 문자열 배열을 `name`, 각 사람의 추억 점수가 담긴 정수 배열을 `yearning`이라고 한다.
- 각 사진에 찍힌 사람들의 이름이 담긴 2차원 문자열 배열을 `photo`라고 할 때,
  각 사진별로 등장하는 사람들의 추억 점수를 합산한 값을 구한다.
- `name`에 없는 사람은 점수를 0으로 간주한다.
- 각 사진의 합산 점수를 순서대로 배열로 반환한다.

### 접근 방식

- `name[i]`와 `yearning[i]`를 `Map<String, Integer>`에 (이름 → 점수) 형태로 저장한다.
- 각 사진(`photo`)을 순회하면서, 사진에 포함된 이름을 Map에서 조회하여 점수를 누적한다.
- 존재하지 않는 이름은 0점이므로 `getOrDefault(이름, 0)` 방식으로 처리한다.

### 왜 이런 방식으로 풀었나?

- `photo`에 있는 각 이름의 점수를 반복해서 찾아야 하므로, 배열에서 매번 선형 탐색을 하면 비효율적이다.
- Map을 사용하면 이름을 키로 점수를 평균 O(1)에 조회할 수 있어 전체 시간 복잡도를 줄일 수 있다.
- 결과적으로 `photo` 전체를 한 번 순회하면서 빠르게 합산할 수 있다.
