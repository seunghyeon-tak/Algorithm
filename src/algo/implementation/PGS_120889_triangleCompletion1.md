## 삼각형의 완성조건(1)

### 문제 요약

- 세 변의 길이가 담긴 배열 sides가 있음.
- 가장 긴 변의 길이는 다른 두 변의 길이 합보다 작다
- 세변으로 삼각형을 이룰 수 있으면 1, 아니면 2를 출력

### 접근 방식

- 한 번의 순회로 전체 합 `sum`과 최대값 `max`를 구한다.
- sum - max > max 이면 1, 아니면 2를 반환

### 왜 이런 방식으로 풀었나?

- 가장 긴 변 vs 나머지 두 변의 합 비교이므로 sum과 max만 있으면 즉시 판별 가능
- 변수가 적고 조건식이 문제 서술과 직결되어 가독성이 높아서

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- Arrays.sort() 방식으로 풀기
  - 정렬 후 s[2] < s[0] + s[1]는 매우 직관적이라 가독성이 좋다.
  - 정렬 비용이 O(n log n) 다만 n=3이라 성능차이는 없음.
