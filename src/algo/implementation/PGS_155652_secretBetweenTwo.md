## 둘만의 암호

### 문제 요약

- 문자열 s, skip와 자연수 index가 주어진다.
- 문자열 s의 각 문자를 기준으로 알파벳을 index만큼 뒤로 이동한다.
- 단, skip에 포함된 알파벳은 이동 과정에서 제외한다.
- skip을 제외한 알파벳만 카운트하여 index번 이동한 결과를 문자열로 만들어 반환한다.

### 접근 방식

- 문자열 s를 순회하면서 문자 하나씩 처리한다.
- 각 문자에 대해 현재 위치에서 한 칸씩 알파벳을 이동한다.
  - 이동 중 'z'를 넘으면 다시 'a'로 순환한다.
  - 이동한 문자가 skip에 포함되어 있으면 카운트하지 않고 넘어간다.
- skip에 포함되지 않은 문자만 카운트하여 index번 도달했을 때의 문자를 결과 문자열에 추가한다.

### 왜 이런 방식으로 풀었나?

- 알파벳을 한 칸씩 이동하며 조건을 검사해야하는 문제라고 생각했다.
- skip 문자는 이동 과정에서 아예 카운트 대상이 아니므로 한 칸씩 이동하여 검사 후 카운트라는 순서를 명확히 유지하는 방식이 가장 직관적이라고 생각했다.
