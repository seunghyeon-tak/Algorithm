## 바탕화면 정리

### 문제 요약

- 문자열 배열 `wallpaper`가 주어진다.
- `#`은 파일이 존재하는 칸, `.`은 빈 칸을 의미한다.
- 모든 파일을 포함하도록 드래그할 때,
  드래그 영역의 좌상단과 우하단 좌표를 `[lux, luy, rdx, rdy]` 형태로 구한다.
- 이때 `rdx`, `rdy`는 **영역의 끝 좌표(포함 X)** 를 의미한다.

### 접근 방식

- 이 문제의 핵심은 파일 하나하나의 좌표가 아니라 파일들을 전부 감싸는 최소 직사각형 영역을 구하는것이다.
- 모든 # 좌표에 대해
  - 행(row)의 최솟값 / 최댓값
  - 열(col)의 최솟값 / 최댓값 을 갱신한다.
- 좌상단 좌표는 (최소 row, 최소 col)
- 우하단 좌표는 영역의 끝 좌표 이므로 (최대 row + 1, 최대 col + 1)로 계산한다.

### 왜 이런 방식으로 풀었나?

- 문자열 인덱스로 얻는 좌표는 점(포인트)이다.
- 문제에서 요구하는 결과는 드래그 영역이다.
- 영역의 끝 좌표는 항상 마지막 점의 좌표 + 1이 된다.
