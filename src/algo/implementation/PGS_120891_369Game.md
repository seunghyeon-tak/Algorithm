## 369게임

### 문제 요약

- 3, 6, 9가 들어간 수를 체크하는 문제
- 정수 order 매개변수가 주어지는데 해당 수에 해당하는 박수 갯수를 리턴하기 

### 접근 방식

- 정수를 문자열로 변환한 뒤, 각 문자에 대해 `'3'`, `'6'`, `'9'`인지 검사하며 카운트했습니다.

### 왜 이런 방식으로 풀었나?

- 문자열로 바꾸면 자리수에 상관없이 **단순 반복문 한 번으로 처리**할 수 있어 구현이 가장 쉽고 직관적이기 때문입니다.

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 수학적으로 나누기/나머지 연산을 반복하는 방식도 가능하지만 문자열 방식이 더 간단하고 가독성이 좋음
- 스트림 활용법
```java
static int solution(int order) {
    return (int) String.valueOf(order)
            .chars()
            .filter(c -> c == '3' || c == '6' || c == '9')
            .count();
}
```
