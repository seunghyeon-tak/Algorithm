## 공원 산책

### 문제 요약

- 2차원 격자 형태의 공원이 주어진다.
- 시작 위치는 `S`, 장애물은 `X`, 이동 가능한 곳은 `O`로 표시된다.
- `routes`에 주어진 방향(N, S, E, W)과 이동 거리만큼 이동한다.
- 단, 이동 중 범위를 벗어나거나 장애물을 만나면 해당 명령은 무효가 된다.
- 모든 이동을 마친 뒤 최종 좌표를 반환하는 문제이다. 

### 접근 방식

1. 시작 위치 `S`를 먼저 찾았다.
2. 장애물 `X`의 좌표를 따로 저장하려고 했다.
3. 방향에 따라 조건을 나누어 이동을 처리하려고 했다.

예시 사고 흐름:

- E면 y를 증가
- W면 y를 감소
- S면 x 증가
- N면 x 감소
- 범위 체크는 if문으로 분기

즉, 방향마다 분기 처리하는 방식으로 접근했다.

### 왜 이런 방식으로 풀었나?

- 장애물이 있을 때 어떻게 처리해야 할지 막혔다.
- “최종 위치만 검사하면 되는 건가?”
- “중간에 X가 있으면 어떻게 하지?”
- 조건을 계속 나누다 보니 코드가 복잡해졌다.

(수정)

1. 방향 벡터 사용
2. 이동을 한 칸씩 검사
3. 이동 가능할 때만 최종 위치 갱신

- 방향마다 분기 처리  
- 장애물 좌표를 따로 저장 가 아니라

- 방향 벡터로 이동량 통일  
- park 배열을 직접 조회  
- 이동 중 하나라도 조건 위반 시 취소
