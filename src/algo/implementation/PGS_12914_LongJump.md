## 멀리 뛰기

### 문제 요약

- 1칸 또는 2칸을 뛸 수 있는데 멀리뛰기에 사용될 n개의 칸수에 맞춰 뛸 수 있는 경우의 수를 구하시오.

### 접근 방식

- dp를 사용하여 접근
- i칸에 도달하는 방법은
  - i - 1칸에서 1칸 뛰어오르는 경우
  - i - 2칸에서 2칸 뛰어오르는 경우

### 왜 이런 방식으로 풀었나?

- 이전 결과를 이용해 현재 결과를 만들 수 있는 구조가 필요하다고 생각함
- 경우의 수를 직접 나열하는 방식은 n이 커질수록 비효율적이기에
  이미 계산한 값을 저장하면서 재사용하는 dp 방식이 더 효율적이다.
