## 문자열 내 마음대로 정렬하기

### 문제 요약

- 문자열로 구성된 stringss 배열과 정수 n이 주어진다.
- 문자열 인덱스 n번째 글자를 기준으로 오름차순 정렬한 배열을 출력하기

### 접근 방식

- Arrays.sort()와 comparator를 사용해 정렬 기준을 직접 정의했다.
- 문자열 a, b를 비교할때
  - 먼저 a.charAt(n)과 b.charAt(n)을 비교한다.
  - n번째 문자가 같다면 a.comparator(b)를 사용해 문자열 전체를 사전순으로 비교한다.

### 왜 이런 방식으로 풀었나?

- Arrays.sort()에 comparator를 넘기면
  - 1순위 기준
  - 동일할때 2순위 기준을 한번에 처리가능하다
- 문제의 조건을 그대로 코드로 옮기기 쉬워서 가장 직관적인 방식이다.
