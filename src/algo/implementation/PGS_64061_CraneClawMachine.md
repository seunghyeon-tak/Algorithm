## 크레인 인형뽑기

### 문제 요약

- 2차원 배열 board로 인형이 쌓여 있다.
  - 0은 빈칸, 숫자는 인형 종류
- moves 배열에 적힌 순서대로 열을 선택해서 그 열에 가장 위에 있는 인형을 뽑아라
- 뽑은 인형은 바구니에 들어가는데,
  - 바구니의 맨 위 인형과 지금 새로 넣는 인형이 같은 종류면 둘 다 터져서 사라짐
- 최종적으로 사라진 인형의 개수를 출력하기

### 접근 방식

- moves 배열의 값들만 잘 활용하여 board의 2차월 배열에 board[move - 1]형태로 접근 가능하다
- 여기서 stack을 사용하여 최근에 넣은 인형과 지금 넣을 인형을 비교해 가며 제거 시켜주면 된다.

### 왜 이런 방식으로 풀었나?

- 결국 마지막에 꺼낸 인형과 지금 넣을 인형이 어떤건지가 제일 중요하다고 생각해 stack이 제일 직관적일것 같아서 해당 방식 사용
