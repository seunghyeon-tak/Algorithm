## 기사단원의 무기

### 문제 요약

- 1번부터 number까지의 각 숫자에 대해 약수의 개수를 구한다.
- 해당 숫자의 약수 개수가 limit보다 크면 power 값을 사용한다.
- 그렇지 않으면 약수의 개수를 그대로 사용한다.
- 최종적으로 모든 값을 더한 결과를 반환한다.

### 접근 방식

- 1부터 number까지 반복하면서 각 숫자의 약수 개수를 계산한다.
- 약수의 개수를 계산할때 시간 초과를 방지하기 위해 1부터 해당 숫자까지 전부 나누지 않고
  제곱근 (i * i) 기준으로 약수를 계산한다.
- 계산된 약수 개수에 따라 limit 조건을 적용하여 answer에 누적한다.

### 왜 이런 방식으로 풀었나?

- 처음에는 1부터 자기 자신까지 전부 나눠보는 방식으로 약수의 개수를 구했다.
- 하지만 이 방식은 number가 커질수록 반복 횟수가 급격히 증가하여 시간 초과가 발생했다.
- 약수는 항상 쌍으로 존재한다는 점에 착안하여 제곱근 까지만 검사하는 방식으로 반복 횟수를 줄였다.

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 좋은 점
  - 약수 계산 범위를 제곱근까지만 줄여 시간 복잡도를 크게 개선할 수 있다.
  - number가 커져도 안정적으로 통과할 수 있다.
- 안좋은 점
  - 처음 접하면 i*i 약수 쌍 개념이 직관적으로 이해되지 않아 구현 난이도가 높게 느껴질 수 있다
