## 실패율

### 문제 요약

- 게임 스테이지가 1번부터 N번까지 있다.
- stages 배열에는 각 사용자가 현재 머물러 있는 스테이지 번호가 있다.
- 실패율이 높은 스테이지부터 정렬
  - 실패율이 같으면 번호 작은 스테이지 먼저 입력
- 스테이지 실패율 계산법
  - 그 스테이지에 머물러 있는 사람 수 / 그 스테이지에 도달한 사람 수

### 접근 방식

- 각 인덱스마다 클리어한 스테이지가 다르기에 배열 하나를 만들어서 스테이지 클리어별로 카운팅으로 정리
- 실패율 배열을 생성 후 실패율 계산을 하여 실패율 배열에 저장
- 실패율을 미리 정렬하게 되면 어떤 플레이어의 실패율이 높았는지 정리가 안되기에 새로운 배열을 하나 생성한다.
- 새로운 배열을 정렬할때 실패율 배열을 비교하여 인덱스 정리를 한다.

### 왜 이런 방식으로 풀었나?

- 한꺼번에 해결하려는 방법보다 하나씩 풀어서 해결하는 방법이 직관적이기 때문에 이렇게 나눠서 풀이함
