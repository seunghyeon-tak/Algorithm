## 그림 확대

### 문제 요약

- 직사각형 형태의 그림 파일이 있다 해당 그림은 1 x 1 크기의 정사각형 크기의 픽셀로 이루어져 있다.
- 그림파일은 문자열 배열 picture와 정수 k가 주어지면, 각 문자를 가로로 k배, 각 행을 세로로 k배 확장한 문자열 배열을 반환한다.

### 접근 방식

- String.valueOf(c).repeat(k)로 문자를 문자열로 변환·반복해 가로로 늘린 뒤, 그 한 줄을 k회 복제해 세로로 확장했습니다.

### 왜 이런 방식으로 풀었나?

- 단순하게 문자열로 인한 내장 함수를 사용해서 푸는게 제일 낫다고 생각해서 해당 방식으로 풀었음
