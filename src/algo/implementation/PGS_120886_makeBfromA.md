## A로 B만들기

### 문제 요약

- 문자열 `before`의 문자들을 재배열하여 문자열 `after`를 만들 수 있는지 판별하는 문제.
- **before를 단순히 뒤집는(reverse) 문제가 아니라**,  
  **두 문자열이 같은 문자들을 동일한 개수만큼 포함하는지**를 검사하는 문제이다.

### 접근 방식

- 두 문자열을 문자 배열로 변환한 뒤 정렬한다.
- 정렬된 결과가 동일하다면, 두 문자열은 같은 문자들을 같은 개수로 포함한 것이므로 after를 만들 수 있다.
- 다르면 재배열로 만들 수 없으므로 0을 반환한다.

### 왜 이런 방식으로 풀었나?

- 문자열을 정렬하면 **문자 순서가 사라지고 문자 구성만 남기 때문에**,  
  애너그램(anagram) 판별에 가장 직관적이고 구현이 간단한 방식이다.
- 추가적인 자료구조가 필요 없고, 비교도 쉬워 효율적이다.
