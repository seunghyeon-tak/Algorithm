## JadenCase 문자열 만들기

### 문제 요약

- JadenCase란 각 단어의 첫 문자는 대문자, 나머지 문자는 모두 소문자로 변환한 문자열이다.
- 단, 단어의 첫 문자가 숫자인 경우 그대로 유지한다.
- 공백은 하나 이상 연속해서 등장할 수 있으며, 원래 공백 형태를 그대로 유지해야 한다.

### 접근 방식

- split()을 사용하지 않고 문자 단위로 순회하였다.
- 현재 문자가 단어의 시작인지 여부를 boolean 변수로 관리하였다.
- 공백을 만나면 다음 문자를 단어 시작으로 처리하도록 설정하였다.
- 단어의 시작 문자라면:
  - 숫자는 그대로 유지
  - 문자는 대문자로 변환
- 그 외의 문자는 모두 소문자로 변환하여 추가하였다.

### 왜 이런 방식으로 풀었나?

- split(" ")을 사용할 경우
  - 연속된 공백 처리에 문제가 발생할 수 있고
  - 원본 문자열의 공백 개수를 그대로 유지하기 어렵다.
- 따라서 문자열을 문자 단위로 직접 제어하는 방식이 더 안전하고 직관적이라고 판단하였다.
- 단어의 시작 여부만 관리하면 구현이 비교적 단순해진다.
