## 2차원으로 만들기

### 문제 요약

- 정수배열 num_list와 정수 n이 매개변수로 주어질때
- num_list에서 n개씩 묶어서 2차원 배열로 나타내기

### 접근 방식

- 1차원 배열을 순회하면서 값을 임시 배열(curr)에 채움
- 임시 배열이 n개로 가득 차면 2차원 배열(answer)의 한 행으로 삽입
- 삽입 후 새로운 임시 배열을 생성해 다음 묶음을 채워 나가는 방식

### 왜 이런 방식으로 풀었나?

- 2차원 배열의 한 행을 구성할때 n개 단위로 끊어야 하므로
- 현재 행을 채우는 배열을 이용해 완성된 순간에만 answer에 넣는 구조가 직관적이라 생각했기때문에

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 인덱스 계산 (i / n, i % n)을 활용하는 방법
```java
answer[i/n][i%n] = num_list[i];
```
