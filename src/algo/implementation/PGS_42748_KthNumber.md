## K번째 수

### 문제 요약

- 배열 array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하려 한다.
- commands 배열의 하나의 배열마다 [i, j, k]의 값이 담겨있다.
- 첫번째는 array의 배열의 i~j까지의 배열을 뽑고
- 두번째로 뽑은 배열을 오름차순으로 정렬 후
- 세번째로 k번째 배열의 값을 뽑은 값을 추출 하기

### 접근 방식

- 배열을 특정 범위로 자르기 위해서 Arrays.copyOfRange()를 사용하였고
- sort()로 오름차순 정렬을 하고 k번째에 값의 뽑아 새로운 배열에 넣어주었다.
