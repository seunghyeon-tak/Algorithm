## 대소문자 바꿔서 출력하기

### 문제 요약

- 영어 알파벳으로 이루어진 str 문자열이 있는데, 각 알파벳을 대문자는 소문자 소문자는 대문자로 바꿔서 출력하기

### 접근 방식

- 입력받은 문자열을 문자 배열(char[])로 변환
- 각 문자를 순회하면서 반대로 변환
- 변환된 문자를 문자열에 다시 넣고 출력

### 왜 이런 방식으로 풀었나?

- `toCharArray()`를 이용하면 문자열을 문자단위로 쉽게 접근 할 수 있어서
- `Character` 클래스의 메서드를 활용하면 대소문자 판별과 변환을 명확하게 할 수 있어서
- StringBuilder는 문자열을 반복해서 더할때 String 보다 메모리 낭비가 적고 효율적이기 때문에

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- String.replaceAll() + 정규식
  - 코드가 짧은 장점이 있지만 대소문자 혼합 문자열엔 비효율적이다.
- stream() + map()
  - 함수형 스타일이고 가독성이 좋다 하지만 현재 문제에선 오버엔지니어링이며 성능적으로 손해
