## 붕대 감기

### 문제 요약

- 붕대 감기 기술은 1초마다 일정 체력을 회복한다.
- 연속으로 t초 동안 붕대를 감는데 성공하면 추가 회복이 발생한다.
- 몬스터에게 공격을 받는 순간:
    - 해당 초에는 회복하지 못한다.
    - 붕대 연속 성공 횟수는 0이 된다.
- 체력은 최대 체력을 초과할 수 없다.
- 체력이 0 이하가 되면 즉시 사망하며 -1을 반환한다.
- 모든 공격이 끝난 뒤의 최종 체력을 구하는 문제이다.

### 접근 방식

- 시간을 1초부터 마지막 공격 시간까지 1초씩 증가시키며 시뮬레이션한다.
- 현재 시간이 공격 시간과 같다면:
    - 해당 데미지만큼 체력을 감소시킨다.
    - 붕대 연속 성공 횟수를 0으로 초기화한다.
- 공격이 없는 시간이라면:
    - 1초 회복량만큼 체력을 증가시킨다.
    - 연속 성공 횟수를 증가시킨다.
    - 연속 성공 횟수가 t에 도달하면 추가 회복을 적용하고 0으로 초기화한다.
- 체력이 최대 체력을 초과하지 않도록 제한한다.
- 체력이 0 이하가 되는 순간 즉시 -1을 반환한다.

### 왜 이런 방식으로 풀었나?

- 이 문제는 시간의 흐름에 따라 상태가 변하는 전형적인 시뮬레이션 문제이다.
- 매 초마다 “공격 여부 → 회복 여부 → 상태 갱신”의 순서를 그대로 구현하는 것이 가장 직관적이고 실수를 줄일 수 있다.
- 복잡한 자료구조나 알고리즘이 필요하지 않고, 문제 조건을 그대로 코드로 옮기는 것이 핵심이다.
