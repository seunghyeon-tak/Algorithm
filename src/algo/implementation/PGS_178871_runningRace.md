## 달리기 경주

### 문제 요약

- 선수들이 현재 등수 순서대로 players 배열에 저장되어 있다.
- 해설진이 특정 선수 이름을 callings 배열을 통해 부른다.
- 이름이 불릴 때마다 해당 선수는 바로 앞 선수 한 명을 추월 한다.
- 모든 호출이 끝난 뒤 최종 선수 순서를 반환하는 문제이다.

### 접근 방식

- Hashmap을 사용하여 선수 이름 : 현재 인덱스로 저장한다.
- callings를 순회하면서
  - 현재 선수의 인덱스를 O(1)로 조회
  - 바로 앞 선수와 swap
  - swap 후 변경된 인덱스를 map에 다시 업데이트

### 왜 이런 방식으로 풀었나?

- 초기에는 단순 배열 탐색 방식으로 구현했는데 해당 방법은 시간 복잡도가 너무 커지기에 시간 초과 에러가 발생하여서 hashmap 방식으로 풀었음
