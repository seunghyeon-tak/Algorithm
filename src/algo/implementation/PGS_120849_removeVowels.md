## 모음 제거

### 문제 요약

- 영어에서 a, e, i, o, u는 모음으로 분류됨
- 문자열 my_string이 있을때 모음을 제거한 문자열을 리턴하시오.

### 접근 방식

- 모음이 담긴 문자 배열을 담고 반복문을 돌면서 해당 문자가 있을땐 break 처리해서 구분

### 왜 이런 방식으로 풀었나?

- 문제의 내용을 직관적으로만 봤을때 문자열의 문자 하나하나 체크해야하는게 제일 직관적일것 같아서

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 정규식 `replaceAll`
  - 장점 : 짧고 직관적, 대소문자 (?i)로 한번에 처리
  - 단점 : 정규식 파싱 비용이 있어 짧은 문자열 반복 호출 시 상대적으로 느릴 수 있음
```java
return my_string.replaceAll("(?i)[aeiou]", "");
```
