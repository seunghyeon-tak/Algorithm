## 두 개 뽑아서 더하기

### 문제 요약

- 정수 배열 numbers가 주어진다.
- numbers안에 서로 다른 두개의 값을 뽑아 더해서 만들 수 있는 모든 수를 배열에 넣고 출력하기

### 접근 방식

- numbers안의 값들을 일단 for문으로 돌려서 모두 더한다 (겹치는거 상관없이)
- HashSet을 사용하여 겹치는 수는 중복 제거 처리후 stream을 사용해서 정수형 배열에 넣어주는 방식 사용

### 왜 이런 방식으로 풀었나?

- 단순히 더하고 set에 넣기만하는 방식이 단순하고 제일 직관적이기때문에 해당 방법 사용
