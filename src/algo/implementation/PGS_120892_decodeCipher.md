## 암호해독

### 문제 요약

- 암호화된 문자열 cipher가 있는데 그 문자열에서 code의 배수 번째 글자만 진짜 암호
- 즉, code번째에 있는 암호만 따로 빼서 출력하기

### 접근 방식

- 반복문을 사용해서 해당 위치(code)에 있는 문자만 출력해서 StringBuilder에 추가해주는 방식 사용

### 왜 이런 방식으로 풀었나?

- substring 방식도 고려했지만, 해당 방식을 이용하면 내부적으로 새로문 문자열을 만들어서 리턴하기때문에 불필요한 객체 생성 비용이 추가되기때문이다.

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 현재 방법이 가장 좋은 형태인것 같다.
