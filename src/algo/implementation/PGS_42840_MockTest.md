## 모의고사

### 문제 요약

- 3명의 수포자가 각자 반복되는 방식으로 문제를 찍는다.
- 실제 정답 배열 answers가 주어질때
- 가장 많이 맞힌 사람의 번호를 오름차순으로 반환한다.
- 공동 1등이면 여러 명을 함께 반환한다.

### 접근 방식

- 각 수포자의 찍는 패턴을 배열로 저장
- 문제를 하나씩 순회하면서 현재 문제 번호 i에 대해 각 수포자의 패턴 값을 i % 패턴길이로 순환해서 가져온다.
- 실제 정답과 비교해서 각 수포자의 맞힌 개수를 카운트 한다.
- 세 사람 중 최댓값을 구한 뒤, 그 점수와 같은 사람들을 결과 리스트에 넣는다.

### 왜 이런 방식으로 풀었나?

- 각 수포자의 답안은 일정한 패턴이 반복되므로, % 패턴길이를 사용하면 문제 번호에 맞는 답안을 쉽게 구할 수 있다.
- 문제 수 만큼 한 번만 순회하면 되므로 구현이 단순하고 직관적이다.
- 최댓값을 먼저 구하고 같은 점수를 가진 사람을 모두 추가하면 공동 1등도 자연스럽게 처리 가능하다.
