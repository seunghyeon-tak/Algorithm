## 동영상 재생기

### 문제 요약

- 동영상 길이, 현재 위치, 오프닝 시작/끝 시간이 주어진다.
- "prev" 명령은 10초 뒤로 이동
- "next" 명령은 10초 앞으로 이동
- 영상 범위를 벗어나면 0초 또는 영상 길이로 보정
- 현재 위치가 오프닝 구간에 포함되면 자동으로 오프닝 끝으로 이동
- 모든 명령 수행 후 최종 위치를 "mm:ss" 형식으로 반환 

### 접근 방식

1. 시간 문자열을 초 단위로 변환하여 처리
2. 명령 수행 시
    - 이동
    - 범위 보정
    - 오프닝 구간 체크
      순서로 처리
3. 최종 초 단위를 다시 "mm:ss" 형식으로 변환

### 왜 이런 방식으로 풀었나?

- 문자열 상태에서 직접 처리하면 복잡해지므로 초 단위 정수로 변환
- 상태 변화 흐름을 명확히 하기 위해 기능별 메서드 분리
- 구현 문제에서 중요한 것은 "처리 순서"이므로
  이동 → 범위보정 → 특수구간 처리 순서를 유지
