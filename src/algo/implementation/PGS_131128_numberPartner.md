## 숫자 짝꿍

### 문제 요약

- X, Y 문자열로된 정수가 주어진다.
- 각각의 두 정수의 문자는 0 <= k <= 9까지 나타낼 수 있다.
- 여기서 X와 Y가 공통으로 가진 수를 가지고 만들 수 있는 가장 큰 수를 출력하기
- 단, X,Y가 짝이 존재하지 않는다면 -1을 공통숫자가 0뿐이면 0을 출력하기

### 접근 방식

- X와 Y 각각에 대해 숫자(0~9)의 등장 횟수를 저장하는 정수 배열을 사용한다.
- 두 배열에서 같은 숫자의 공통 개수는 `min(xCount[i], yCount[i])`로 구할 수 있다.
  - 공통으로 사용할 수 있는 개수는 더 적은 쪽에 맞춰야 하기 때문이다.
- 가장 큰 수를 만들기 위해 9부터 0까지 역순으로 순회하며,
  공통 개수만큼 해당 숫자를 결과 문자열에 추가한다.

### 왜 이런 방식으로 풀었나?

- 문자열에서 직접 문자를 제거하거나 비교하는 방식은 시간 복잡도가 커질 수 있다고 판단했다.
- 숫자의 범위가 0~9로 고정되어 있기 때문에, 카운팅 배열을 사용하면 문제를 단순화할 수 있다.
- 각 숫자의 공통 개수만 알면 결과를 바로 구성할 수 있어 구현과 로직이 명확해진다.
