## 구슬을 나누는 경우의 수

### 문제 요약

- balls : 전체 구슬의 개수
- share : 선택할 구슬의 개수
- balls개의 구슬 중에서 share개의 구슬을 고르는 모든 조합의 수를 구하는 문제

### 접근 방식

- 조합의 기본 공식은 nCr = n! / (r! * (n - r)!)
- 하지만 n!은 수가 매우 커져서 int형에서 오버플로우가 발생할 수 있음
- 따라서 팩토리얼 전체를 계산하지 않고, 약분 후 남는 부분을 단계별로 계산
  - n * (n - 1) * (n - 2) * ... * (n - r - 1)

### 왜 이런 방식으로 풀었나?

- 팩토리얼 전체를 그대로 계산하면 n!이 빠르게 int형 범위를 초과하기 때문
- 조합식을 단계별로 계산하면 정확도와 효율성을 모두 확보 가능
