## 약수 구하기

### 문제 요약

- 매개변수 n이 주어짐
- n을 나누었을때 나머지가 0이 되는 모든 정수를 찾아 배열로 반환하는 문제

### 접근 방식

- 약수를 저장할 ArrayList<>를 준비하고
- 1부터 n까지 반복하면서 n % i == 0인 i를 리스트에 추가
- 마지막에 리스트를 int 배열로 변환하여 반환

### 왜 이런 방식으로 풀었나?

- 로직이 단순하고 가독성이 좋아 이해하기 쉽다.
- 약수의 정의를 그대로 코드로 옮긴 방식이라 직관적임

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 스트림으로 간결하게 구현하는 방식

```java
import java.util.stream.IntStream;
return IntStream.rangeClosed(1, n).filter(i -> n % i == 0).toArray();
```
