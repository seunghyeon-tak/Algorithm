## 자릿수 더하기

### 문제 요약

- 자연수 `n`의 각 자릿수를 모두 더한 값을 반환하는 문제.
- 예: n = 987 → 9 + 8 + 7 = 24

### 접근 방식

- 정수를 문자열로 변환한 뒤 문자 배열(char[])로 변환한다.
- 각 문자에서 '0'을 빼 정수로 변환하여 모두 더한다.

### 왜 이런 방식으로 풀었나?

- 문자열 변환 방식은 구현이 가장 직관적이며,
  추가적인 수학적 계산 없이 곧바로 각 자릿수를 추출할 수 있다.
- 가독성도 좋고 실수할 여지가 적다.

### 다른 방식으로 풀었을때 좋은점? (혹은 안좋은점)

- 수학적 연산 사용 (%10, /10) 방식
```java
int sum = 0;
while (n > 0 ){
    sum += n % 10;
    n /= 10;
}
```
- 장점 : 문자열 변환없이 정수 연산만 사용하므로 성능이 더 좋음
- 단점 : 어떨땐 직관적이지 않을 수 있으며, 나눗셈 반복 구조를 이해해야함
