## 그리디 알고리즘

> 매 순간 가장 좋아 보이는 선택을 하는 알고리즘

### 예시로 이해해보기

- 거스름돈 문제
  - 거스름돈 : 1260원, 동전 : 500, 100, 50, 10
  - 가장 큰 동전부터 쓰면? 총 6개
    - 500 -> 2개
    - 100 -> 2개
    - 50 -> 1개
    - 10 -> 1개

### 특징

- 지금 가장 좋은 선택
- 그걸 계속 반복

### 문제 특징

- 정렬 후 선택
- 가장 큰 것 / 가장 작은 것 선택
- 한 번 선택하면 되돌릴 필요 없는 문제
- 목표를 빨리 채우는 문제

### 주의

- 그리디는 모든 문제에서 되는게 아님
